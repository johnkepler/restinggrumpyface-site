Accessibility

yaAnnotator works with macOS's built-in accessibility features — turn them on and the app responds automatically.

yaAnnotator works with macOS's built-in accessibility features. Turn them on in System Settings ▸ Accessibility — the app responds automatically, with nothing to enable inside yaAnnotator.

VoiceOver

Every control, image, and badge has a spoken label. Toggle VoiceOver with ⌘F5.

  • Image list and grid — each thumbnail reads as one element: its name, annotation count, and approval / issue state.
  • The canvas — the annotation surface reports the selected shape, its label, and its position.
  • Report charts — the bar, distribution, and scatter charts expose an audio graph and a Details rotor, so you can scrub a distribution and hear its shape; every bar and point also has a spoken label and value, and the centroid heatmap carries a summary description. Anything a chart does when tapped is mirrored in the toolbar's Filter menu, so it's reachable without a pointer.
  • Live status — analysis starting or finishing, and a report filter being applied or cleared, are announced as they happen.

Keyboard navigation

The List and Single-column image views are fully keyboard-navigable: arrow keys move the selection, Return opens an image, Delete removes it, and Esc deselects. (The Grid is pointer-driven — use List or Single-column for keyboard-only work.) The annotation tools and every menu command have shortcuts; see Keyboard Shortcuts.

Appearance and color

  • Dark / Light — choose System, Light, or Dark in Settings ▸ General (⌘,). The whole app, including the canvas, follows it.
  • Legible badges — the annotation-count, size, and issue pills use fills tuned to stay readable against any thumbnail, in light or dark mode.
  • Color is never the only cue — issue severity is shown with an icon and a count, not color alone, and the report charts respond to Differentiate Without Color. Label colors come from a colorblind-safe palette and can be changed per class in the Label Manager (⇧⌘L).

Reduced motion

With Reduce Motion on, the app drops non-essential animation — scrolling the selected image into view, and the editor's panel reveal, become instant.

Voice Control

Menus, toolbars, sheets, and the List views work with Voice Control's spoken names and dictation. Dictation is a quick way to type class labels and fill in Notes hands-free.
